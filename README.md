# Login & Register UI
A beautiful login and registration page built with HTML, CSS and JavaScript.

## Live Demo
https://rajeshwari2805.github.io/login-register/login-register.html

## Features
- Clean modern UI design
- Sign In and Create Account tabs
- Password strength meter
- Form validation
- Social login buttons (Google & GitHub)
- Responsive design

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Note
This repository contains the frontend UI only.
The backend (PHP + MySQL) runs locally using XAMPP.
Full backend files available in the project folder.

## How to Run Locally
1. Install XAMPP
2. Copy project to `C:\xampp\htdocs\`
3. Start Apache and MySQL in XAMPP
4. Open browser and go to:
   `http://localhost:8080/login-register/login-register.html`
